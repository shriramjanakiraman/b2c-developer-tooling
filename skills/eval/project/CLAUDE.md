# B2C Commerce Storefront

This is a Salesforce B2C Commerce (SFCC) storefront project using SFRA (Storefront Reference Architecture).

## Stack

- Salesforce B2C Commerce Cloud (Demandware)
- SFRA controllers and ISML templates
- B2C CLI (`@salesforce/b2c-cli`) for development tooling
- PWA Kit / Storefront Next for headless frontend (planned)
